# Voice Emotion Analyzer 🎤

A Deep Learning-based system that detects human emotions from speech audio using MFCC features and CNN.

## 🔍 Features
- Real-time emotion prediction
- MFCC feature extraction
- CNN model (.h5)
- Interactive frontend using HTML, CSS, JS

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- Librosa
- NumPy
- HTML, CSS, JavaScript

## 📂 Project Structure
- speech_emotion_model.h5 → trained model  
- live_prediction.ipynb → prediction notebook  
- final_check.py → execution script  
- templates/index.html → frontend  
- css/, js/ → styling & logic  

## 📊 Dataset
RAVDESS dataset (not included due to large size)

## 🚀 How to Run
1. Install dependencies  
2. Run `live_prediction.ipynb` or `final_check.py`  
3. Open frontend
